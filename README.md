# FinTrack

## Giới thiệu
FinTrack là ứng dụng giúp người dùng quản lý tài chính cá nhân:
- Ghi chép thu – chi
- Phân loại chi tiêu
- Thiết lập ngân sách
- Xem báo cáo tài chính

## Mục tiêu
- Quản lý chi tiêu hiệu quả
- Giao diện dễ dùng
- Dữ liệu an toàn

## Phương pháp
- Agile – Scrum
- Phát triển theo Sprint
